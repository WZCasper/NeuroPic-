## Что такое NeuroPic?

NeuroPic — это бесплатный веб-сайт для генерации изображений с помощью искусственного интеллекта. Работает прямо в браузере, без регистрации и без оплаты.

### Возможности

| Функция | Описание |
|---|---|
| ✦ Бесплатная генерация | Pollinations AI — 5 моделей без ключей |
| ✦ Ваши ключи | Hugging Face, Replicate — дополнительные модели |
| ✦ Автовыбор модели | Система сама выбирает лучшую доступную модель |
| ✦ История | Сохраняется в браузере, не исчезает при перезагрузке |
| ✦ 10 стилей | Авто, Фото, Живопись, Аниме, Киберпанк и др. |
| ✦ 4 соотношения | 1:1, 16:9, 9:16, 4:3 |
| ✦ 1–4 изображения | Несколько вариантов за раз |
| ✦ Лайтбокс | Просмотр и навигация между изображениями |
| ✦ Скачивание | Одно или все сразу |
| ✦ Тёмная/светлая тема | Сохраняется между сессиями |
| ✦ Улучшение промпта | Автоматическое добавление качественных тегов |
| ✦ Негативный промпт | Расширенные настройки |
| ✦ Сид | Воспроизведение точного результата |

## Как разместить на GitHub Pages (пошагово)

### Шаг 1 — Создать репозиторий

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт
2. Нажмите зелёную кнопку **New** (или **+** → **New repository**)
3. Назовите репозиторий: `NeuroPic`
4. Выберите **Public** (обязательно — для бесплатного хостинга)
5. Нажмите **Create repository**

### Шаг 2 — Загрузить файлы

**Способ А — через браузер (без команд):**

1. В созданном репозитории нажмите **uploading an existing file**
2. Перетащите все файлы из папки NeuroPic:
   - `index.html`
   - `style.css`
   - `app.js`
   - `.github/workflows/deploy.yml` (создайте через "Create new file")
3. Напишите в поле commit: `Initial upload`
4. Нажмите **Commit changes**

**Способ Б — через Git:**
```bash
cd NeuroPic
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/NeuroPic.git
git push -u origin main
```

### Шаг 3 — Включить GitHub Pages

1. В репозитории перейдите в **Settings** (вкладка вверху)
2. В левом меню найдите **Pages**
3. В разделе **Source** выберите **GitHub Actions**
4. Сохраните

### Шаг 4 — Готово!

Через 2–3 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/NeuroPic
```

## Файл для папки .github/workflows/

Если загружаете через браузер, создайте файл вручную:

1. Нажмите **Create new file**
2. В поле имени напишите: `.github/workflows/deploy.yml`
3. Скопируйте содержимое файла `deploy.yml` из этой папки
4. Нажмите **Commit new file**
